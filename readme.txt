Woocommerce whole enquiry cart plugin let your customers to send their entire cart as an email to you inbox.

== Description ==

With whole enquiry cart plugin you can let your customers send their entire cart as email to you. This plugin will add a new button in cart page where your customers can send their enquiry with all the products in the cart.

It's not like other enquiry cart plugins. Most of other plugin uses separate cart for enquiries. But whole enquiry cart uses same WooCommerce Cart and customers still be able to add items to cart and checkout. Sometimes customers might need to ask some enquiries about items in the cart, so they can simply use the options giving by this plugin to send entire cart directly to admin's inbox.

Woocommerce is required.


== Frequently Asked Questions ==

= Can I change the email address to receive the enquiry details? =

Yes you can!. In the plugin settings you can add your email.

= Do I need to install Woocommerce to use this plugin? =

Yes Woocommerce is required to be installed.

= Can I use this to get an enquiry about single product? =

This is only for cart page. There will is no option developed yet to add enquiry for single product page.

= I am not receiving email, what is wrong? =

Please check plugin settings and confirm if you have added the email address there.
